PROJECT TITLE : Student Marks Analyzer
Description :
Beginner friendly data analysis project that analyzes student marks using python

what it does :
- reads the data from the csv file (marks details)
- calculates the total and average marks
- then it assigns grades according to the marks
- visualizes the perfomance of students with a bar graph

libraries used :
[ Python,Pandas,Matplotlib ]

Steps to run :
-install dependencies
-run the mark_analyzer.ipynb

Output :
Displays the bar chart containing marks and names of students and a table with all the mark details including average and total marks
